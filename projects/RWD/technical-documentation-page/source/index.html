Introduction to CSS Flexbox

Flexbox is a layout module in CSS that provides a flexible and efficient way to layout items within a container.

It simplifies creating responsive layouts that adapt to different screen sizes.

display: flex;
  • Key feature: Flexibility
  • Key feature: Responsiveness

Getting Started with Flexbox

To use Flexbox, set the container's display property to flex.

Example:

.container { display: flex; }
  • Step 1: Define a flex container
  • Step 2: Style flex items

Flex Container Properties

Properties like flex-direction control the layout of items.

Example properties:

flex-direction: row;
  • flex-direction
  • justify-content
  • align-items

Flex Item Properties

Properties like order control the item's position.

Example:

order: 2;
  • order
  • flex-grow
  • flex-shrink

Common Use Cases and Examples

Flexbox is used for navigation menus and equal-height columns.

Example:

nav { display: flex; }
  • Responsive navigation
  • Equal-height columns
  • Centering elements
